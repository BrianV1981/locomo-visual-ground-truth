# Images Directory

This directory is the local archive of all surviving image assets from the original LoCoMo dataset.

*   Contains 775 locally preserved, downscaled (Fair Use) `.jpg` files.
*   The `locomo_v2_local.json` variant maps its URLs directly to this folder, enabling air-gapped enterprise evaluation.
*   The `locomo_v2_web.json` variant maps to the raw GitHub URLs of these exact files.
